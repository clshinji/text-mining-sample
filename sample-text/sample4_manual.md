# 自動改札機システム保守マニュアル

## 文書管理情報
- 文書番号: TM-AGM-001-Rev.3
- 作成日: 2025年7月10日
- 承認者: 技術部長 鈴木太郎
- 適用範囲: 全駅自動改札機（AG-2000シリーズ）

## 1. 概要

本マニュアルは、駅務機器として設置されている自動改札機AG-2000シリーズの日常保守作業について規定する。対象機器は以下の通りである。

- AG-2001型: 標準改札機（幅員550mm）
- AG-2002型: 車椅子対応改札機（幅員900mm）
- AG-2003型: 高速処理改札機（処理速度60人/分）

## 2. 日常点検手順

### 2.1 始業前点検（所要時間：15分）

**手順1: 電源系統確認**
1. 主電源スイッチがON位置にあることを確認
2. UPS（無停電電源装置）の状態表示LEDが緑色点灯していることを確認
3. 電圧計で入力電圧がAC100V±10%の範囲内であることを測定

**手順2: 機械系統点検**
1. 改札扉の開閉動作を手動で3回実施
2. 扉センサーの反応を確認（遮光時に扉が閉まることを確認）
3. 券投入口に異物がないことを目視確認
4. 券排出口の清掃を実施

**手順3: 通信系統確認**
1. 中央制御装置との通信状態を確認
2. ネットワーク接続ランプが正常点灯していることを確認
3. 時刻同期が正常に行われていることをシステム画面で確認

### 2.2 IC読取装置点検

**必要工具:**
- 専用清拭布（型番：CL-001）
- イソプロピルアルコール（濃度99.5%以上）
- テスト用ICカード（管理番号：TEST-001）

**作業手順:**
1. IC読取部表面を専用清拭布で清拭
2. テスト用ICカードでタッチ動作を10回実施
3. 読取エラー率が1%以下であることを確認
4. アンテナ感度が-70dBm以上であることを測定器で確認

## 3. 定期保守作業

### 3.1 週次保守（毎週月曜日実施）

**券詰まり防止保守:**
- 券搬送ローラーの清掃
- 券ガイドの位置調整
- 券検知センサーの感度確認

**所要時間:** 1台あたり30分
**実施者:** 機械部門技術員（資格：改札機保守技能士2級以上）

### 3.2 月次保守

**主要部品交換基準:**
- 券搬送ローラー: 100万回通過または6ヶ月
- ICアンテナ: 信号強度-75dBm以下
- 扉駆動モーター: 異音発生時または50万回動作

## 4. 故障対応

### 4.1 エラーコード一覧

| コード | 内容 | 対応方法 |
|--------|------|----------|
| E001 | 券詰まり | 券除去後、リセットボタン押下 |
| E002 | IC読取エラー | アンテナ清拭、感度調整 |
| E003 | 扉動作異常 | センサー確認、手動リセット |
| E004 | 通信異常 | ネットワーク接続確認 |

### 4.2 緊急時対応

**完全停止時の手順:**
1. 手動開放レバーを操作し、利用者の通行を確保
2. 故障表示板を設置
3. 中央制御室に故障報告（内線番号：2001）
4. 保守業者への連絡（24時間対応：0120-XXX-XXX）

## 5. 安全注意事項

- 保守作業時は必ず「保守中」表示を設置
- 電気系統作業時は絶縁手袋を着用
- 可動部品の点検時は電源を切断
- 作業完了後は必ず動作確認を実施

## 6. 記録・報告

日常点検結果は「改札機保守記録簿」（様式第3号）に記録し、月末に技術部へ提出する。異常発見時は直ちに「設備故障報告書」（様式第7号）を作成し、部門長の承認を得ること。
